About CustomMobs
CustomMobs is a Minecraft plugin that allows anyone to create special mobs and bosses using an easy to use in-game GUI. This GUI allows :
  • Changing armor and held item(s) of the mob.
  • Setting the name and type of the mob.
  • Editing entity options like behavior or attributes.
  • Customizing the drops the mob gives on death.
  • Configuring spawner settings like range and speed.
  • Adding custom effects like timed events or abilities.
  • Modifying sounds and apply potion effects.
  • Setting messages for spawn and death events.
    • The CustomMobs can then be spawned using an egg (or the customized spawning item) or with a spawner.
